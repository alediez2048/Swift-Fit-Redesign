## Typography

### Headlines
**Font Family:** Playfair Display (or similar sophisticated Serif)
**Characteristics:** Elegant, editorial, timeless

### Body Text
**Font Family:** Inter (or similar clean Sans-Serif)
**Characteristics:** Readable, modern, clean

### Usage Guidelines

```
H1: Playfair Display, 48-64px
H2: Playfair Display, 36-48px
H3: Playfair Display, 24-32px
Body: Inter, 16-18px
Caption: Inter, 14px
```

---

## Design Principles

1. **Breathe** - Generous whitespace, let elements breathe
2. **Warmth** - Inviting, not cold or clinical
3. **Sophistication** - Elevated, luxury feel without pretension
4. **Authenticity** - Reflects Austin's unique character
5. **Clarity** - Clean hierarchy, easy navigation
